# Another Farm — Game Homepage

Official marketing website and privacy policy for **Another Farm**, a cozy mobile farming simulation game available on iOS.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Main homepage with game overview and features |
| `privacy.html` | Privacy policy (English) |
| `oauth_callback.html` | OAuth callback handler for sign-in flow |

## About the Game

Another Farm is a relaxing mobile farming game where players can:

- Grow a variety of crops and flowers
- Raise animals and collect resources
- Customize and expand their farm
- Sync progress across devices via Google or Apple account

Available on the **iOS App Store**.

## Tech Stack

- Pure HTML + CSS, no build tools required
- Self-contained static site, deployable anywhere

## Deployment

```bash
# Preview locally
npx serve .
# or
python3 -m http.server
```

Deploy to GitHub Pages, Netlify, or any static host. The site is pre-configured for Google Search Console verification.
